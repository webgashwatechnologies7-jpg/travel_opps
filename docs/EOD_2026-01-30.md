## Summary
Itinerary detail page par user-reported issues fix kiye: **Package Terms** click ab kaam karta hai, aur suggested items add karte waqt **day select na hone** par clear message show hota hai.

## 1. Package Terms – Click Functionality

- **Issue:** Left sidebar mein "Package Terms" (DAY 4 ke neeche) par click karne par kuch nahi ho raha tha.
- **Fix:**
  - Package Terms div par `onClick` add kiya → click par **Final** tab open hota hai.
  - Keyboard support: `role="button"`, `tabIndex={0}`, aur `onKeyDown` (Enter/Space) se bhi Final tab open ho sakta hai.
- **File:** `frontend/src/pages/ItineraryDetail.jsx`

---

## 2. Suggested Itinerary Items – User Feedback (Day Not Selected)

- **Issue:** Right sidebar se Day Itinerary / Activity add karte waqt agar user ne pehle koi **day select nahi kiya** (e.g. DAY 1, DAY 2), to koi feedback nahi aa raha tha – user ko lagta tha button kaam nahi kar raha.
- **Fix:**
  - **Day Itinerary** – Card click aur **+** button dono par: agar `selectedDay` nahi hai to alert: *"Please select a day from the left (e.g. DAY 1, DAY 2) first, then add this item."*
  - **Activity** – Card click aur **+** button dono par: same message add kiya.
- **Note:** Accommodation (Hotel) add ke liye pehle se hi "Please select a day first" message tha – koi change nahi.
- **File:** `frontend/src/pages/ItineraryDetail.jsx`
